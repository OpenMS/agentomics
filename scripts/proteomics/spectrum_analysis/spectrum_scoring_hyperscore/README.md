# Spectrum Scoring HyperScore

Score experimental spectrum against theoretical using a HyperScore-like approach.

## Usage

```bash
python spectrum_scoring_hyperscore.py --mz-list "100.5,200.3,300.1" --intensities "1000,500,200" --sequence PEPTIDEK --charge 2
python spectrum_scoring_hyperscore.py --mz-list "100.5,200.3" --intensities "1000,500" --sequence PEPTIDEK --charge 1 --output score.json
```
